This document explains the purpose of each folder and file in the project directory:

-Controllers
Contains the controller classes that handle user interactions and serve as a bridge between the Fxml and Model layers.

-Data
Stores data-related files, such as the database or configuration files.

-Fxml
Contains the .fxml files that define the structure and layout of the user interface (UI).

-Models
contains classes,core logic and data structure of the application (business logic).

-Views
controllers of the GUI like , which element to show when.

-App
the entry point of the application.
